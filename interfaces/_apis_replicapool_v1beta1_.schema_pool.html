autoRestart: boolean
  • Defined in apis/replicapool/v1beta1.ts:318

Whether replicas in this pool should be restarted if they experience a failure. The default value is true.

baseInstanceName

baseInstanceName: string
  • Defined in apis/replicapool/v1beta1.ts:328

The base instance name to use for the replicas in this pool. This must match the regex a-z?. If specified, the instances in this replica pool will be named in the format <base-instance-name>-<ID>. The <ID> postfix will be a four character alphanumeric identifier generated by the service. If this is not specified by the user, a random base instance name is generated by the service.

targetPools

targetPools: string[]
  • Defined in apis/replicapool/v1beta1.ts:380

A list of target pools to update with the replicas that are managed by this pool. If specified, the replicas in this replica pool will be added to the specified target pools for load balancing purposes. The replica pool must live in the same region as the specified target pools. These values must be the target pool resource names, and not fully qualified URLs.
