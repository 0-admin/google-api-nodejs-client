name: string

usePrivateAddress

usePrivateAddress: boolean
  • Defined in apis/genomics/v2alpha1.ts:424

If set to true, do not attach a public IP address to the VM. Note that without an public IP address, additional configuration is required to allow the VM to access Google services. See https://cloud.google.com/vpc/docs/configure-private-google-access for more information.
