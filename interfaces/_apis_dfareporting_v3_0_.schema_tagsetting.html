additionalKeyValues

additionalKeyValues: string
  • Defined in apis/dfareporting/v3.0.ts:6863

Additional key-values to be included in tags. Each key-value pair must be of the form key=value, and pairs must be separated by a semicolon (;). Keys and values must not contain commas. For example, id=2;color=red is a valid value for this field.

includeClickThroughUrls

includeClickThroughUrls: boolean
  • Defined in apis/dfareporting/v3.0.ts:6868

Whether static landing page URLs should be included in the tags. This setting applies only to placements.

keywordOption

keywordOption: string
  • Defined in apis/dfareporting/v3.0.ts:6878

Option specifying how keywords are embedded in ad tags. This setting can be used to specify whether keyword placeholders are inserted in placement tags for this site. Publishers can then add keywords to those placeholders.
