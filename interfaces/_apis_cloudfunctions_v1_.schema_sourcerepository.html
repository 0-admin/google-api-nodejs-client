deployedUrl

deployedUrl: string
  • Defined in apis/cloudfunctions/v1.ts:445

Output only. The URL pointing to the hosted repository where the function were defined at the time of deployment. It always points to a specific commit in the format described above.

url

url: string
  • Defined in apis/cloudfunctions/v1.ts:458

The URL pointing to the hosted repository where the function is defined. There are supported Cloud Source Repository URLs in the following formats: To refer to a specific commit: https://source.developers.google.com/projects/x/repos/x/revisions/x/paths/x To refer to a moveable alias (branch):https://source.developers.google.com/projects/x/repos/x/moveable-aliases/x/paths/x In particular, to refer to HEAD use master moveable alias. To refer to a specific fixed alias (tag): https://source.developers.google.com/projects/x/repos/x/fixed-aliases/x/paths/x You may omitpaths/x if you want to use the main directory.
