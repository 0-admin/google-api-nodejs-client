displayName

displayName: string
  • Defined in apis/cloudresourcemanager/v2.ts:169

The folder’s display name. A folder’s display name must be unique amongst its siblings, e.g. no two folders with the same parent can share the same display name. The display name must start and end with a letter or digit, may contain letters, digits, spaces, hyphens and underscores and can be no longer than 30 characters. This is captured by the regular expression: \p{L}\p{N}?.

lifecycleState

lifecycleState: string
  • Defined in apis/cloudresourcemanager/v2.ts:174

Output only. The lifecycle state of the folder. Updates to the lifecycle_state must be performed via DeleteFolder and UndeleteFolder.
