archiveUris

archiveUris: string[]
  • Defined in apis/dataproc/v1.ts:945

Optional. HCFS URIs of archives to be extracted in the working directory of Spark drivers and tasks. Supported file types: .jar, .tar, .tar.gz, .tgz, and .zip.

args

args: string[]
  • Defined in apis/dataproc/v1.ts:951

Optional. The arguments to pass to the driver. Do not include arguments, such as --conf, that can be set as job properties, since a collision may occur that causes an incorrect job submission.

fileUris

fileUris: string[]
  • Defined in apis/dataproc/v1.ts:956

Optional. HCFS URIs of files to be copied to the working directory of Spark drivers and distributed tasks. Useful for naively parallel tasks.

jarFileUris

jarFileUris: string[]
  • Defined in apis/dataproc/v1.ts:961

Optional. HCFS URIs of jar files to add to the CLASSPATHs of the Spark driver and tasks.

loggingConfig

loggingConfig: Schema$LoggingConfig
  • Defined in apis/dataproc/v1.ts:965

Optional. The runtime log config for job execution.

mainClass

mainClass: string
  • Defined in apis/dataproc/v1.ts:970

The name of the driver's main class. The jar file that contains the class must be in the default CLASSPATH or specified in jar_file_uris.

mainJarFileUri

mainJarFileUri: string
  • Defined in apis/dataproc/v1.ts:974

properties

properties: any
